### User Story 1:
As a new user, I want to register for an account with a secure login system that requires both email and password so that I can access my account securely.

* Acceptance Criteria:
  • The system must require both email and password to complete the registration process.
  • The system must display an error message if the email or password is invalid or not provided.
  • The system must generate a unique and secure password for the new user.

### User Story 2:
As an existing user, I want to log in to my account using the secure login system that requires both email and password so that I can access my account and perform tasks securely.

* Acceptance Criteria:
  • The system must prompt the user to enter their email and password for login.
  • The system must validate the email and password credentials and display an error message if they are incorrect.
  • The system must display a successful login message after correct credentials are entered.

### User Story 3:
As an administrator, I want to manage user accounts with the secure login system that requires both email and password so that I can ensure the security and integrity of user data.

* Acceptance Criteria:
  • The system must allow administrators to view and edit user account information.
  • The system must generate a password reset link for users who forget their password.
  • The system must display a list of all users with their respective login credentials.
