Project: Event Management System

User Story
As an organizer I want to create events so that users can register for them.

Acceptance Criteria
- Event creation form available
- Organizer authentication required
- Event details saved in database
- System response < 2 seconds


User Story
As a user I want to register for events so that I can attend them.

Acceptance Criteria
- Registration button available
- User confirmation message shown
- Event capacity tracked
- System response < 2 seconds
