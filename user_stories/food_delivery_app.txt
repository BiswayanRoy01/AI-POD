Project: Food Delivery App

User Story
As a user I want to browse restaurants so that I can order food online.

Acceptance Criteria
- Restaurant list displayed
- Search option available
- Menu accessible from restaurant page
- System response < 2 seconds


User Story
As a user I want to place food orders so that I can receive meals at my location.

Acceptance Criteria
- Add to cart option available
- Checkout process enabled
- Order confirmation displayed
- System response < 2 seconds
